Overview

We are a competitive minecraft server. If you like building bases, blowing up others, and pvp you will like our server.

Features

  • Factions - Get a team together and fight against other teams.
  • Web server - Access our server from school or work where normal game servers are blocked.
  • No pay2win

Join

  • Minecraft: factions.lethalmc.net
  • Discord: discord.lethalmc.net

Support

Lethal MC is ran by poor college students, and mainaing a server isn't free. Server costs go over $10 a month plus we have put in many free hours developing and improving the server. You can support us by donating. All donations go towards server costs and improvements. People who consistently support the server through donations, playing, testing, and other ways will be given VIP rank. Note: this is not a one-time donation rank that you purchase. We reserve the right to change your status on our server at any time. If you would like to support our server, you can cover a month of our expenses below.
