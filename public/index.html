Overview

We are a competitive minecraft server. If you like building bases, blowing up others, and pvp you will like our server.

Features

  • Factions - Get a team together and fight against other teams.
  • Web server - Access our server from school or work where normal game servers are blocked.
  • No pay2win

Join

  • Minecraft: factions.lethalmc.net
  • Discord: discord.lethalmc.net

Support

Lethal MC is a non-commercial server ran by poor college students, and running it isn't free. Server costs go over $10 a month, and we have put in many free hours developing and improving the server. You can support us by donating, playing, testing, voting, and other ways. All donations go towards server costs and improvements. People who consistently support will receive VIP rank.
